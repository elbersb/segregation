# Student-level data including SES status

Fake dataset used for examples. This is an individual-level dataset of
students in schools.

## Usage

``` r
school_ses
```

## Format

A data frame with 5,153 rows and 3 variables:

- school_id:

  school ID

- ethnic_group:

  one of A, B, or C

- ses_quintile:

  SES of the student (1 = lowest, 5 = highest)
